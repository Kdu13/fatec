HOME IF SWITCH FOR WHILE ARRAY

IF

Executa um bloco de código se a condição for verdadeira.

IF...ELSE

Executa um bloco se a condição for verdadeira, senão executa outro bloco.

ELSE IF

Permite testar múltiplas condições em sequência.

SWITCH

Compara uma variável com vários valores e executa o bloco correspondente.

FOR

Laço de repetição com número definido de repetições.

WHILE

Executa o bloco enquanto a condição for verdadeira.

ARRAY

Estrutura que armazena vários valores em uma única variável.
